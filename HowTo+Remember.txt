------------How To-----------------
-App.vue 파일에서 Coding 이 이루어짐
-npm run serve 를 통해 localhost://???? 서버 생성 후 ctrl+LClick 통해 확인




-------------Remember-----------------
-App.vue 즉 .vue 확장자 파일은 컴퓨터가 열 수 없음 대신에 이를 index.html 안으로 
App.vue 를 컴파일시켜서 다 때려박아서 html 형태로 우리눈에 보여짐
- html 로 박아넣는 작업을 main.js 에서 하고있음.

App.Vue 파일에서 데이터바인딩 시 
data()
return {} <-- return 의 중괄호 내부에 필요한 데이터를 이름,데이터값 (price1 : 70, )이런식으로 등록하고
그대로 상위 <template> 의 내부 <div> 에서 데이터바인딩 즉 {{}} 중괄호 2개(콧수염) 내부에서 
내가 지정한 데이터의 이름만 가져와서 사용하면 됨 ex) {{price1}} 

{{데이터바인딩}} 하는 이유
1. HTML 에 그대로 {{price1}} 대신에 {{70}}으로 하드코딩 해두면 나중에 변경이 어려움
2. Vue 의 실시간 자동 렌더링 쓰려면 하셈 --> Vue 는 신기해서 아래에서 data 를 변경하면 data 와 관련된 HTMl 에도 실시간으로 바로 반영
